# N + 1 queries problems
Gặp khi ta query list id rồi query detail cho từng id, mỗi id là 1 câu lệnh SQL. Vấn đề này luôn có thể giải quyết bằng cách sửa code. 
Giải pháp: Gom thành 1 query với JOIN; Dùng 2 queries, 1 query lấy list id, 1 query dùng IN (list_id). Sau đó xử lý thêm ở trong code cho data trả về.
